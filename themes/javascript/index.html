Editing your Themes Javascript

When you are working with JavaScript you need to make sure to place any code within our $Ready function. This function is called whenever a page is loaded and is fired when the document is ready.

We use jQuery as our main library of choice.

$Ready(function() {
	// add JS here... jQuery is supported!
});
