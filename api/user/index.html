Activity Feed API

Access to activity feed. Used to add any base content for the site and is open to custom fields, which is based on passing JSON. Each feed is connected to the Like & Comment app.

/user

Gets an array of users.

GET

Returnsarray

Key Info
id (int) ID of the user.
name (string) First and Last name.
url (string) URL to the users profile.
gender
id: ID of gender.
name: Phrase for gender.
photo
50px: 50px profile photo.
120px: 120px profile photo.
200px: 200px profile photo.
original: Original profile photo.
